# Project: Sicong for Unqork

Static HTML portfolio site. Single `index.html` with CSS and vanilla JS.

## Structure
- `index.html` — the entire site (5 pages: About, Experience, Process, Collaboration, Q&A)
- `public/photos/` — all photo assets

## Design Tokens (CSS custom properties in :root)
- **Colors**: navy, purple, teal, pink, surface variants, on-surface variants, outline variants
- **Fonts**: Newsreader (serif headings), Manrope (sans-serif body/labels)
- **Labels**: 10px uppercase, letter-spacing 0.2em, Manrope 700

## Deployment
- Vercel static site — serves `index.html` from root
- Photos referenced as `public/photos/filename.ext`
